# 🌌 Code Archeology

Explore GitHub developers like ancient civilizations.  
Each repository is a city, commits are artifacts, and programming languages are the culture of the developer civilization. 🏙️💻

---

## 🛠 Features

- 🔍 Search any GitHub username  
- 🏙️ Visualize repositories as "cities" on an interactive globe  
- 📊 View developer stats: repositories, stars, main language, rank  
- 🌌 Animated 3D globe using **Three.js**  
- ⚡ Responsive and clean UI  

---

## 💡 Concept

- **Repositories** → Cities  
- **Commits** → Artifacts  
- **Stars** → Power of the civilization  
- **Languages** → Culture / DNA  

---

## 🖥 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Visualization:** Three.js, GSAP  
- **API:** GitHub REST API  
- **Dev Server:** Vite
